# Login Signup Dashboard

A modern Login, Signup and Dashboard UI built using:

- Node.js
- Express
- MongoDB
- Handlebars (HBS)
- HTML, CSS, JavaScript

## Features
- User authentication
- Glassmorphism UI
- Client-side validation
- Responsive dashboard design

## How to Run
1. Clone the repo  
   `git clone https://github.com/Anupamvats52/login-signup-dashboard`
2. Install dependencies  
   `npm install`
3. Start server  
   `node src/index.js`
4. Open browser  
   `http://localhost:3000`
